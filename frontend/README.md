## Phase 7: Backend & Frontend Integration

This phase focuses on integrating the backend APIs with the frontend UI and running the complete application locally.

## Project Setup Guide

### 1. Clone the Repository

```bash
git clone git@github.com:vstech-solutions/dec2025-internship-batch.git
```

### 2. Frontend Setup

- Navigate to the frontend directory and install dependencies:

```bash
cd krushna-ghuge/frontend
npm install
npm run dev
```

- Frontend will run at: http://localhost:5173

### 3. Backend Setup

- Open a new terminal and navigate to the backend directory:

```bash
cd krushna-ghuge/backend
npm install
npm run dev
```

- Backend server will start on `5001` port.

### 4. Access the Application

- Open your browser and visit : `http://localhost:5173/ `
